The Google Fonts API

Description

The Google Fonts API is an accessible and easy-to-use tool which allows you to withdraw a font from Google's servers for use on your own website. All fonts are free for commercial use and there are a wide arrange of styles available. Google Fonts hosts over 1000 different fonts, each with several weights and styles. The Google Fonts API is sure to host a font which is applicable to your project.
